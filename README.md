# Press or Perish

Roguelike journalism simulation set in 1970s rural Sweden. You play as a reporter at a small-town newspaper, navigating ethical dilemmas while trying to survive five days without getting fired.

## Live Demo
[→ Play Press or Perish](https://press-or-perish.web.app)

## Features
- Papers Please-inspired pixel art aesthetic
- Nordic noir storytelling with branching choices
- NPC interactions, boss dialogue, and reputation system
- Multiple story paths and endings

## Tech Stack
- Vanilla JavaScript / HTML / CSS
- JSON-driven dialogue and story data
- Firebase Hosting

## Running Locally
Open `index.html` in a browser — no build step required.
